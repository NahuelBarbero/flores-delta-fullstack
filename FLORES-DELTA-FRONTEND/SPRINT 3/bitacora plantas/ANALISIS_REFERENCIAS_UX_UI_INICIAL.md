# Análisis Inicial de Referencias UX/UI para Flores Delta (Basado en Imágenes)

**Propósito:** Documentar el análisis visual inicial de las plataformas de referencia (Cannect, Growdiaries, Grow with Jane) para extraer patrones de UX/UI, paletas de colores, tipografías y funcionalidades clave que servirán como inspiración y guía para el diseño de Flores Delta.

---

## 1. Cannect (Estética "Tech" y Sofisticada)

### 1.1. Paleta de Colores:
*   **Dominante:** Tonos oscuros de azul/verde.
*   **Acentos:** Verde brillante para CTAs y elementos gráficos.
*   **Texto:** Blanco/gris claro.
*   **Insight:** El cliente mencionó que le gusta el "color de cannect".

### 1.2. Tipografía:
*   Sans-serif moderna y audaz para titulares, ligera para cuerpo de texto.

### 1.3. Diseño y Composición:
*   **Sección Hero:** Título principal impactante, visualización de datos (Temperatura, Luminosidad, Humedad) en círculos prominentes con gráficos abstractos.
*   **Beneficios:** Presentación radial con iconos y texto conciso.
*   **Trazabilidad:** Diagrama de flujo vertical para etapas de vida de la planta (Semilla, Esqueje, Flor Seca) con imágenes y etiquetas. **Directamente relevante para la trazabilidad semanal del cultivo.**
*   **Gestión en Tiempo Real:** Cuadrícula de tarjetas/módulos con icono y título.
*   **Login:** Minimalista, iconos en campos, visibilidad de contraseña.
*   **Navegación:** Barra superior clara, botón "Ingresar" destacado, chat flotante persistente.

---

## 2. Growdiaries (Formato de "Diario de Cultivo" y Comunidad)

### 2.1. Paleta de Colores:
*   Verdes y gris/blanco, más orgánica.

### 2.2. Navegación Principal:
*   Sidebar izquierdo con iconos.
*   Top Bar con logo, búsqueda, botones de acción.

### 2.3. Feed de Contenido (Diarios de Cultivo):
*   Cuadrícula de tarjetas con imagen de planta, nombre, tiempo de cultivo, metadatos.
*   Barra de filtros horizontal por categorías.
*   **Insight:** Formato de "tarjetas" y filtros **altamente relevante** para la bitácora de cultivo de Flores Delta.

### 2.4. Login:
*   Campos estándar, icono de "ojo".
*   **Opciones de Autenticación Social (SSO):** Botones prominentes con iconos y colores de marca.

### 2.5. Explorar Cepas:
*   **Filtrado Avanzado:** Fila superior de tarjetas con iconos/etiquetas, barra de búsqueda, múltiples filtros desplegables.
*   **Catálogo:** Cuadrícula de tarjetas con imagen de cepa, nombre, calificación, métricas (THC, CBD).
    *   **Insight:** Patrón de filtrado y catálogo **altamente relevante** para genéticas y productos.

---

## 3. Grow with Jane (Estilo "Amigable" y Móvil-First)

### 3.1. Paleta de Colores:
*   Fondo blanco, verde vibrante para logo, titulares y elementos gráficos.

### 3.2. Ilustraciones:
*   Estilo "línea" o "dibujo" con toques de verde, carácter amigable y accesible.

### 3.3. Interfaz de Usuario Móvil (Captura de Pantalla):
*   **Calendario Mensual:** Con días resaltados. **Directamente relevante para la trazabilidad semanal.**
*   **Toolbox:** Fila de iconos para acciones rápidas (Action, Photo, Tree log, Environment log).
*   **Journal:** Sección para feed de actividades.
*   **Insight:** Patrones claros para calendario de cultivo y "toolbox" de acciones rápidas, fundamental para la experiencia móvil.

### 3.4. Secciones Educativa y de Comunidad:
*   Tarjetas de guía de cultivo.
*   Cuadrícula de "Growlogs" destacados.
*   **Insight:** Relevante para la idea de comunidad y contenido generado por el usuario.

---

## 4. Conclusiones Generales para Flores Delta (Síntesis Inicial)

*   **Paleta de Colores:** Combinar la sofisticación "tech" de Cannect con la frescura de Grow with Jane o la simplicidad de Growdiaries.
*   **Trazabilidad y Bitácora:** Adoptar el diagrama de flujo de etapas de Cannect y el calendario/journal de Grow with Jane, complementado con el formato de tarjetas de Growdiaries.
*   **Navegación:** Considerar sidebar (Growdiaries) o barra superior limpia, con fuerte componente móvil.
*   **Formularios:** Iconos en campos, visibilidad de contraseña, explorar SSO.
*   **Catálogo/Genéticas:** Utilizar patrón de tarjetas y filtros avanzados de Growdiaries.
*   **Comunidad/Educación:** Integrar elementos de contenido generado por el usuario y guías educativas.
*   **Mobile-First:** Priorizar diseño responsivo y experiencia móvil.
